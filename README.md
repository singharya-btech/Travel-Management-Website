# Travel Management Website

A full-stack MERN travel management application with Docker support.

## Features

* User authentication
* Travel package management
* Booking system
* Responsive frontend
* REST API backend
* Dockerized deployment

## Tech Stack

### Frontend

* React.js
* Axios
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### DevOps

* Docker
* Docker Compose

# Installation

## Clone Repository

```bash
git clone https://github.com/singharya-btech/Travel-Management-Website.git
```

---

# Run Frontend

```bash
cd frontend
npm install
npm start
```

---

# Run Backend

```bash
cd backend
npm install
npm start
```

---

# Docker Deployment

```bash
docker-compose up --build
```

---

# Environment Variables

Create `.env` file inside backend:

```env
MONGO_URI=your_mongodb_url
PORT=5000
```
